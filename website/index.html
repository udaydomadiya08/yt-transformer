YT Downloader

Download YouTube videos in one command — MP4, MP3, subtitles, metadata & comments

Quick start — one line, any OS:

bash <(curl -s /dl) "https://youtu.be/..."
Requires Python 3.8+ & ffmpeg · Mac: brew install python ffmpeg

Metadata

Title, tags, chapters, heatmap, description

💬

Comments

Playlists

CLI — full control

One-shot (no download)

Runs directly, nothing saved to disk:

bash <(curl -s /dl) "URL"

Install globally

Download once, use anywhere:

sudo curl -sL /dl -o /usr/local/bin/ytdl && sudo chmod +x /usr/local/bin/ytdl

Then just run: ytdl "URL"
