How video creation works

1

Enter a topic or describe the video you want

2

Gemini AI generates a script with per-scene permutations

3

Smart search finds matching YouTube clips with precise timestamps

4

Clips downloaded as 3-4 sec segments (never full videos)

5

AI narration + original audio + music mixed dynamically

6

Final video rendered with transitions — ready for YouTube

Configuration

Gemini API Key

Saved locally in your system — never sent anywhere. 2-3 API calls per video.

Output

Custom output directory. Temp files cleaned automatically after render.

Quality

Up to 1080p H.264 video, 320kbps AAC audio. Same HD quality as yt-dlp.
